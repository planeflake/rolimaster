## List Notes

The list supports covert missions through concealment, attack preparation, ambush improvements, manufactured evidence, and studied-target effects.

## Spell Descriptions

| Level | Spell | Description |
|---:|---|---|
| 1 | Concealed Object I | Camouflages an object no larger than a dagger as part of the caster's clothing or skin until it is moved or the duration expires. |
| 2 | Study Patsy | Records a target's activities for 24 hours. A second casting within the next two hours reveals the recorded information and establishes the studied patsy used by later spells. |
| 3 | Well-Aimed Attack I | Grants the caster's next attack within one minute a `+25` bonus. |
| 4 | Open Ambush I | If the caster's next attack qualifies for flank or rear bonuses, Ambush may be used with a `+1` bonus even when the target is aware of the caster. |
| 5 | Targeting I | Allows the caster's next missile attack within one minute to target a stationary or predictably moving target, adjusting critical location by `1`. |
| 6 | Create Evidence I | Creates one non-mobile piece of physical evidence that implicates the currently studied patsy. |
| 7 | Concealed Object II | As Concealed Object I, but affects an object up to sword size. |
| 8 | Well-Aimed Attack II | As Well-Aimed Attack I, but the bonus is `+50`. |
| 9 | Open Ambush II | As Open Ambush I, but the Ambush modification is `+2`. |
| 10 | Targeting II | As Targeting I, but the critical-location modification is `2`. |
| 11 | Lightning Reactions | Grants `+10` to the caster's initiative rolls for the round. |
| 12 | Create Evidence II | As Create Evidence I, but creates two pieces of evidence, one of which is incriminating. |
| 13 | Well-Aimed Attack III | As Well-Aimed Attack I, but the bonus is `+75`. |
| 14 | Open Ambush III | As Open Ambush I, but the Ambush modification is `+3`. |
| 15 | Targeting III | As Targeting I, but the critical-location modification is `3`. |
| 16 | Concealed Object True | As Concealed Object I, but the concealed object may be any size smaller than the caster. |
| 17 | Study Victim | As Study Patsy, but records the information required by the later Slaying Attack spells. |
| 18 | Create Evidence III | As Create Evidence I, but creates three pieces of evidence, two of which are incriminating. |
| 19 | Well-Aimed Attack True | As Well-Aimed Attack I, but the bonus is `+100`. |
| 20 | Slaying Attack | The caster's next attack within one minute against the studied victim is resolved as a slaying attack. |
| 25 | Open Ambush True | As Open Ambush I, but the Ambush modification is `+5`. |
| 30 | Targeting True | As Targeting I, but the critical-location modification is `5`. |
| 50 | Slaying Attack True | For the duration, all of the caster's attacks against the studied victim are resolved as slaying attacks. |
