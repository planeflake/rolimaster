# Amateur Mage Revised

## Summary

A revised companion version of the spare-time magical student. The amateur mage has studied magic as a hobby and wants to learn more, but has not yet become a true student. Retain [[Amateur Mage]] as the core-rule version.

## Acquisition

- **Type:** Lifestyle (L)
- **Time to acquire:** 70 months
- **Starting money:** Normal

## Special Benefits

| Benefit | Target |
|---|---:|
| Spell adder (`+2`) | 50 |
| Daily II item (casting a 3rd-level spell) | 50 |
| Daily I item (casting a 5th-level spell) | 40 |
| Spell multiplier (`x2`) | 10 |
| Spell adder (`+1`) | 0 |
