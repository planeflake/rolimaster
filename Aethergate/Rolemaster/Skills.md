# Skills

Source material for Rolemaster skills used in Mana World.

## Available Fields

The Skill Companion supports these fields for a full skill import:

| Field | Purpose |
|---|---|
| Skill | Skill name as printed in the source. |
| Category | Skill category or broad rules family. |
| Source | Book and source page. |
| Stat Pair | Governing category stat pair used for skill calculations, sourced from *School of Hard Knocks*. |
| Stat Bonus | Flat stat bonus imported from the workbook for reference. |
| Maneuver Type | Static, moving, resistance, combat, or special handling. |
| DP Cost | Development-point cost or reference cost. |
| Resolution Table | Static maneuver table, moving maneuver table, attack table, or special table used. |
| Time / Pace / Exhaustion | Time required, pace limits, exhaustion costs, or repetition rules where applicable. |
| Tools / Requirements | Required tools, materials, knowledge, training, or circumstances. |
| Summary | Short rules-facing description for quick use. |
| Mana World Notes | Setting adaptation notes for Aethergate and the Nine Fonts. |
| Import Status | `roster`, `draft`, or `complete`. |

## Notes On This Import

The full *School of Hard Knocks: The Skill Companion* skill roster is listed below from the table of contents and cross-filled from `Rolemaster - Character Generation sheet.xls`.

Costs use the workbook's displayed skill-cost columns on the `Skills` tab. `Stat Bonus` keeps the workbook's displayed flat value from its `Stat` column. `Stat Pair` records the governing category pair used for skill calculations, with pairs inferred from the *School of Hard Knocks* optional-stat lines where the first two stats are the category stats.

The Skill Companion roster is a strong base for Aethergate's non-spell mechanics. Power, lore, craft, urban, and occult-adjacent skills can be tied directly to the [[Nine Fonts]], guild training, temple certification, and the social machinery of [[Aethergate]].
